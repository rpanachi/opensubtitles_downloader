OpenSubtitles Downloader
========================

OpenSubtitles.org hash calculator and downloader compatible with mips32 (OpenWRT)

## Installation

* Copy bin/oshash to /bin or add to your PATH

## Usage

```
./osdownloader movie.avi pob
```
